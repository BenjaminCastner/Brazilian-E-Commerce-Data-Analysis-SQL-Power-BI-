# Brazilian E-Commerce Data Analysis (SQL + Power BI)
This project analyses a Brazilian e-commerce dataset to uncover insights into sales performance, customer behaviour, and delivery efficiency.

The analysis was conducted using PostgreSQL for data preparation and Power BI for dashboard visualisation.

##OBJECTIVES
- Analyse revenue trends over time
- Identify key geographic markets
- Evaluate delivery performance and efficiency
- Understand customer order behaviour

##TOOLS USED
- PostgreSQL (data cleaning & transformation)
- Power BI (dashboard & visualisation)
- Excel (light data inspection)

##KEY INSIGHTS
- Revenue increased steadily throughout 2017–2018, indicating strong growth in customer demand.
- São Paulo is the dominant market, contributing the majority of total orders.
- Approximately 89% of deliveries are completed on time, showing strong logistical performance.
- Delivery times are generally short, with a small number of longer delays.
